# Netflix Replica
## Introduction

This is a student project that was created at [CodeOp](http://codeop.tech), a full stack development bootcamp in Barcelona. With it we aim to replicate Netflix web. 

## Technologies used

For this project we have used Visual Studio Code as the code editor, Vue as Javascript framework, Tailwind as CSS framework.

## Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.
